# Simple Weather App (Go)

Это простое веб-приложение, написанное на языке Go, которое позволяет узнать текущую погоду в любом городе мира. Проект выполнен в рамках первой недели стажировки.

## Функционал

- Ввод названия города.
- Отображение текущей температуры в градусах Цельсия.
- Текстовое описание погоды (на русском языке).
- Адаптивный дизайн (корректно отображается на мобильных устройствах).
- Обработка ошибок (например, если город не найден).

## Технологический стек

- **Backend:** Go (Golang) — используется стандартная библиотека `net/http` и `html/template`.
- **Frontend:** HTML, CSS (адаптивная верстка с использованием Flexbox).
- **Данные:** OpenWeatherMap API (Current Weather Data).
