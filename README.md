# Weather App (Go)
Простое приложение для отслеживания погоды в реальном времени.

### Стек:
- Go (стандартная библиотека net/http, html/template)
- CSS (адаптивная верстка)
- OpenWeatherMap API

### Как запустить:
1. Клонируйте репозиторий.
2. Получите API ключ на [OpenWeatherMap](https://openweathermap.org/).
3. Запустите проект: `APP_KEY=ваш_ключ go run main.go`.
4. Откройте в браузере: `http://localhost:8080`.
